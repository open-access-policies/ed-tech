# ANNEX-001: FERPA Control Mapping

## Purpose

This annex maps **[Company Name]**'s policies and controls to the requirements of the Family Educational Rights and Privacy Act (FERPA), 34 CFR Part 99.

## Verification Checklist

Use this checklist to verify FERPA compliance for school engagements:

### Pre-Contract
- [ ] DPA describes educational services (institutional function)
- [ ] DPA establishes school control over data
- [ ] DPA restricts use to specified educational purposes
- [ ] DPA includes audit rights for school

### Ongoing Operations
- [ ] Access limited to legitimate educational interest
- [ ] Disclosure log maintained
- [ ] Subcontractors bound by equivalent terms
- [ ] Amendment requests processed per school direction
- [ ] Access requests supported within 45 days

### Contract Termination
- [ ] Data returned to school (if requested)
- [ ] Data destroyed per schedule
- [ ] Destruction certificate provided
